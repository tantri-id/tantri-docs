# 🔐 Pengaturan PIN

Modul **Pengaturan PIN** memungkinkan kamu menambahkan lapisan keamanan tambahan pada sistem kasir dengan mewajibkan otentikasi PIN untuk tindakan-tindakan tertentu. Fitur ini membantu mencegah perubahan yang tidak sah selama transaksi atau pengelolaan kasir.

---

## ✅ 1. Mengaktifkan PIN

Untuk mulai menggunakan otorisasi berbasis PIN:

1. Buka menu **Pengaturan** → **Atur PIN**.
2. Aktifkan toggle **Aktifkan PIN** di bagian atas halaman.
3. Klik tombol **Simpan** di bagian bawah halaman.

> ℹ️ Setelah PIN diaktifkan, kamu bisa mengatur tindakan-tindakan mana saja yang membutuhkan otentikasi PIN.

---

## ⚙️ 2. Tindakan yang Dapat Diatur dengan PIN

Kamu dapat mengaktifkan otorisasi PIN untuk beberapa tindakan berikut:

| Tindakan                        | Deskripsi                                                                 |
| ------------------------------- | ------------------------------------------------------------------------- |
| **Ubah Metode Pembayaran**      | Memerlukan PIN saat mengubah metode pembayaran pada transaksi.            |
| **Laporan Keuangan**            | Memerlukan PIN saat membuka laporan keuangan dari kasir.                  |
| **Pembatalan Pesanan**          | Memerlukan PIN untuk membatalkan pesanan yang sedang berlangsung.         |
| **Mengurangi/Menghapus Produk** | Memerlukan PIN untuk mengurangi atau menghapus produk dari pesanan.       |
| **Menu Habis / Matikan Produk** | Memerlukan PIN untuk menandai produk sebagai habis atau menonaktifkannya. |
| **Ubah Stok Produk**            | Memerlukan PIN untuk menambah atau mengurangi stok produk.                |
| **Penarikan Saldo Dompet**      | Memerlukan PIN saat melakukan penarikan saldo dari dompet internal.       |

### Cara Mengaktifkan:

1. Aktifkan tombol switch pada setiap tindakan yang ingin kamu lindungi dengan PIN.
2. Klik tombol **Simpan** setelah melakukan pengaturan.
