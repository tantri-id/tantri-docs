Modul **Tipe Order** di Tantri Cashier digunakan untuk mengatur skema pemrosesan pesanan di toko, baik untuk kasir (merchant) maupun pelanggan (customer online). Tipe order ini menentukan apakah pesanan menggunakan sistem **Close Bill** atau **Open Bill**.

---

## ğŸª 1. Tipe Order Merchant

Digunakan untuk menentukan skema order yang digunakan oleh kasir saat membuat transaksi di outlet.

### Pilihan Tipe:

| Tipe Order     | Deskripsi                                                                                                                       |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Close Bill** | Pesanan langsung dibayar dan diselesaikan saat itu juga. Biasanya digunakan untuk sistem kasir cepat.                           |
| **Open Bill**  | Pesanan dapat dibuka terlebih dahulu, kemudian dibayar belakangan. Cocok untuk dine-in, reservasi, atau sistem pencatatan meja. |

### Cara Mengatur:

1. Buka menu **Pengaturan â†’ Tipe Order**.
2. Klik **Tipe Order Merchant**.
3. Pilih salah satu tipe: **Close Bill** atau **Open Bill**.
4. Centang sebagai **Set as Default** jika ingin dijadikan default saat transaksi dimulai.
5. Klik **Simpan**.

---

## ğŸ›’ 2. Tipe Order Customer

Digunakan untuk mengatur tipe order yang tersedia bagi pelanggan yang melakukan **order online** (misal via Tantri Ordering).

### Pilihan Konfigurasi:

| Opsi                 | Deskripsi                                                                       |
| -------------------- | ------------------------------------------------------------------------------- |
| **Semua Tipe Order** | Pelanggan dapat memilih antara open bill atau close bill.                       |
| **Hanya Close Bill** | Semua pesanan online pelanggan wajib langsung diselesaikan (dibayar).           |
| **Hanya Open Bill**  | Pesanan online bisa dicatat dulu dan dibayar belakangan (jika didukung outlet). |

### Cara Mengatur:

1. Masuk ke menu **Pengaturan â†’ Tipe Order â†’ Tipe Order Customer**.
2. Pilih salah satu opsi dari dropdown:
   - Semua Tipe Order
   - Hanya Close Bill
   - Hanya Open Bill
3. Klik **Simpan**.
