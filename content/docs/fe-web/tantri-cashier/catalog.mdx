Modul **Katalog** di Tantri Cashier berfungsi untuk mengelola seluruh data produk atau menu yang ditampilkan di sistem kasir dan pelanggan. Kamu dapat menambahkan produk, mengelompokkan dalam kategori, mengatur status aktif/nonaktif, serta mengunggah menu dari file PDF.

---

## 🧾 1. Struktur Modul Katalog

Terdapat 3 tab utama di halaman Katalog:

1. **Menu Produk**  
   Tempat utama untuk melihat dan mengelola daftar produk/menu.
2. **Varian Produk**  
   Digunakan untuk menambahkan variasi seperti ukuran, topping, level pedas, dll.
3. **Pengaturan Kategori**  
   Digunakan untuk membuat dan mengelompokkan produk berdasarkan kategori tertentu.

---

## 🛠️ 2. Fitur Utama Menu Produk

### ✅ Menambahkan Produk

- Klik tombol **+ Tambah Menu** di pojok kanan bawah.
- Masukkan informasi produk:
  - Nama menu
  - Harga
  - Kategori
  - Deskripsi (opsional)
  - Gambar (opsional)
- Klik **Simpan**.

### 🔄 Status Produk

Gunakan tombol filter dan switch status untuk:

- **Aktif / Nonaktif** → Menampilkan atau menyembunyikan produk dari kasir.
- **Tersedia / Habis** → Menandai ketersediaan stok untuk tiap produk.

### 🔍 Pencarian & Filter

- Gunakan kolom pencarian untuk mencari nama menu tertentu.
- Gunakan filter kategori dan status untuk mempercepat navigasi.

---

## 🗂️ 3. Kategori Produk

Produk dapat dikelompokkan dalam kategori untuk memudahkan pencarian dan pengaturan.

### Menambahkan Kategori:

1. Buka tab **Pengaturan Kategori**.
2. Klik **Tambah Kategori**.
3. Masukkan nama kategori (contoh: Minuman, Makanan Berat, Dessert).
4. Klik Simpan.

> Produk yang tidak memiliki kategori akan otomatis muncul di tab **Tanpa Kategori**.

---

## 📄 4. Unggah File Menu PDF

Tantri Cashier menyediakan fitur untuk **mengunggah file menu dalam format PDF**, sehingga:

- Bisa digunakan sebagai referensi visual saat input manual.
- Memudahkan proses update menu.

### Cara Mengunggah:

1. Klik tombol **Upload PDF Menu**.
2. Pilih file PDF dari perangkat.
3. File akan muncul di atas tab menu.
4. Klik **Hapus PDF** jika ingin mengganti file.

---

## 🔗 5. Integrasi Inventory (Opsional)

Jika menggunakan modul **Inventory**, kamu dapat:

- Mengaktifkan toggle **Integrasi Inventory**.
- Produk akan otomatis terhubung dengan bahan baku yang tercatat di sistem inventori.
