# Table Management

Modul **Table Management** memungkinkan pengguna untuk mengelola tata letak meja di restoran atau kafe secara visual dan fleksibel, mulai dari **aktivasi fitur**, hingga **pembuatan dan pengaturan layout meja**.

---

## 1. Mengaktifkan Fitur Table Management

Sebelum mulai menggunakan fitur ini, **pastikan Table Management diaktifkan terlebih dahulu**:

1. Masuk ke halaman **/settings/table-management**.
2. Aktifkan opsi **Integrasi Table Management**.
3. Aktifkan opsi lainnya sesuai kebutuhan.
4. Klik tombol **Simpan / Save**.

> ⚠️ Catatan: Jika fitur belum diaktifkan, menu "Layout Meja" tidak akan muncul di sidebar Tantri.

---

## 2. Membuka Menu Layout Meja

Setelah fitur diaktifkan:

1. Pergi ke **Dashboard POS**.
2. Klik menu **Table** di sidebar.
3. Kamu akan diarahkan ke editor visual untuk membuat tata letak meja.

---

## 3. Membuat Tata Letak Meja

Untuk membuat grup ataupun tataletak meja dilakukan di **backoffice tantri**.
Di halaman **Pengelolaan Meja**, kamu bisa membuat dan mengatur posisi meja secara visual.

---

## 4. Menggunakan Meja Saat Transaksi

Setelah selesai atau tata letak meja sudah dibuat, kamu bisa menggunakan meja saat membuat order.:

- Di halaman **Table**, saat membuat order baru:
  1. Kamu bisa memilih meja dan pilih opsi buat pesanan baru.
  2. Order akan tercatat berdasarkan meja.
  3. Status meja akan otomatis berubah menjadi "Occupied / Terisi" jika ada order aktif.

---

## 5. Mereset Tata Letak (Opsional)

Jika opsi otomatis bersihkan meja aktif maka meja akan langsung direset ketika transaksi selesai.
Jika ingin mulai ulang:

1. Buka menu **Table** di sidebar.
2. Klik meja yang ingin direset, lalu klik **Bersihkan Meja** (Status pesanan harus "Selesai").
3. Konfirmasi tindakan.

---

## FAQ

### Apakah satu meja bisa digunakan untuk banyak transaksi?

Tidak. Satu meja hanya bisa dipakai untuk satu transaksi aktif pada satu waktu.

### Apakah saya bisa memindahkan posisi meja?

Ya. Cukup drag & drop meja di canvas editor, lalu klik **Simpan Tata Letak** (Di Backoffice).

---

## Integrasi Lanjutan (Opsional)

- Meja dapat dihubungkan dengan sistem **QR Menu / QR Order** untuk pemesanan langsung dari pelanggan.
- Meja juga bisa dihubungkan ke printer khusus berdasarkan zona layout (misal: area A cetak ke dapur 1).
