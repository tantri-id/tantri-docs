This is web application for Tantri Menu which is used for manage order by Menu. This web app created using web builder Vite and ReactJS with programming language typescripts.

So before you start, make sure you have already setup the environment for developing NodeJS Application. You can use any package manger like <b>npm, yarn, pnpm, or bun</b> to install and run this app.

## Techstack :

- ReactJS
- Typescripts
- Shadcn UI
- Tailwind css
- Tanstack React Query
- React Hook Form
- Vite
