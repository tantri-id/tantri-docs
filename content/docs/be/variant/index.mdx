# Endpoint

`GET/product/variant/data`

```ts
https://api.tantri.id/product/variant/data
```

# Deskripsi

Endpoint ini digunakan untuk mengambil semua data variant dari sistem.

```http
Authorization: Bearer <your_token>
```

