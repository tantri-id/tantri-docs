This is web application for Tantri Backend which is used for manage order by Backend. This web app created using Nestjs with programming language typescripts.

So before you start, make sure you have already setup the environment for developing NodeJS Application. You can use any package manger like <b>npm, yarn, pnpm, or bun</b> to install and run this app.

## Techstack :

- Nestjs
- Typescripts
- Postgresql
- Prisma ORM
