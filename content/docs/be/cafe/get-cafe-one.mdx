# Endpoint

`GET/cafe/:id`

```ts
https://api.tantri.id/v2/cafe/:id
```

# Deskripsi

Endpoint ini digunakan untuk mengambil data Cafe dari sistem.

## ðŸ”‘ Autentikasi

Endpoint ini memerlukan autentikasi menggunakan token JWT. Jika pengguna belum login, maka pengguna akan diarahkan ke halaman login.

```http
Authorization: Bearer <your_token>
```
