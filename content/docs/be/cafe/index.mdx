# Endpoint

`GET/cafe`

```ts
https://api.tantri.id/v2/cafe
```

# Deskripsi

Endpoint ini digunakan untuk mengambil semua data Cafe dari sistem.

## Request Params

```ts
{
  id: string;
  search: string;
  sort: string;
  limit: number;
  page: number;
  isDelete: boolean;
  isDiscount: boolean;
}
```
