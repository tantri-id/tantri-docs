## Development

<Callout title="Prerequisite">
  Please install <b>npm</b>
  before proceeding with the installation.
</Callout>

Follow these steps to run this project locally.

<b>Step 1:</b> Clone the repository (Recommended using SSH)

```bash
git clone git@github.com:tantri-id/tantri-backend-ts.git
```

<b>Step 2:</b> Navigate to the project directory

```bash
cd tantri-backend-ts
```

<b>Step 3:</b> Install dependencies

```bash
npm install
```

<b>Step 4:</b>Run development

```bash
npm run dev
```

<b>Step 5:</b> Open browser on localhost:3000

<b>Step 6:</b> after successfully render you can start development.
