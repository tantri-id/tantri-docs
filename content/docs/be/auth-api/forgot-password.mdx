# Endpoint

`POST/auth/cafe/send-forgot-password`

```ts
// General
https://api.tantri.id/auth/cafe/send-forgot-password
```

# Deskripsi

Endpoint ini digunakan untuk mendapatkan OTP Forgot Password.

# Request Body

```json
{
  "email": "jhondoe@gmail.com",
  "portal": "CASHIER"
}
```

# Response Body

```json
{
  "isSuccess": true,
  "message": "Success",
  "data": null
}
```
