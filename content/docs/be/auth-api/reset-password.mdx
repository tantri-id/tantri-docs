# Endpoint

`POST/auth/cafe/reset-password`

```ts

https://api.tantri.id/auth/cafe/reset-password
```

# Deskripsi

Endpoint ini digunakan untuk mereset password.

# Request Body

```json
{
  "email": "string",
  "verificationCode": "string",
  "newPassword": "string"
}
```

# Response Body

```json
{
"isSuccess": boolean,
  "message": "Success",
  "data": null
}
```
