# Endpoint

`POST/auth/cafe/branch`

```ts
https://api.tantri.id/auth / cafe - branch;

```

# Deskripsi

Endpoint ini digunakan untuk memilih cabang cafe.

# Request Body

```json
{
  "idCafe": "string"
}
```

# Response Body

```json
{
  "isSuccess": true,
  "message": "OK",
  "data": {
    "access_token": "string"
  }
}
```
