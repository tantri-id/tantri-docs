## Development

<Callout title="Prerequisite">
  Please install <b>Yarn</b>
  before proceeding with the installation.
</Callout>

Follow these steps to run this project locally.

<Steps>
<Step>
Clone the repository
```bash
git clone git@github.com:tantri-id/tantri-app-react-native.git
```
</Step>

<Step>
Navigate to the project directory

```bash
cd tantri-app-react-native
```

</Step>

<Step>
Install dependencies

```bash
yarn install
```

</Step>

<Step>
Install patch for dependencies

```bash
yarn postinstall
```

</Step>

<Step>
Start the Metro bundler

```bash
yarn start
```

</Step>

<Step>Open your Android Emulator or iOS Simulator</Step>

<Step>
Start on Android Emulator or iOS Simulator

```bash
yarn android    # for android
yarn ios        # for ios
```

</Step>

<Step>Open your IDE and Start Development!</Step>

</Steps>

## Production

Building app apk or aab for android

### Cleaning Gradle Build

```bash
cd android/
./gradlew clean
```

### Building APK Files

```bash
cd android/
./gradlew assembleRelease
```

### Building AAB File

```bash
cd android/
./gradlew bundleRelease
```
