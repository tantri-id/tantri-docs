This is mobile application for Tantri POS which is used to manage the transactions. This application created using React Native CLI.

So before you start, make sure you have already setup the environment for develop React Native Project. You can visit [React Native Setup Environment](https://reactnative.dev/docs/set-up-your-environment) page to set up your environment.
