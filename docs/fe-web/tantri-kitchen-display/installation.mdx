## Development

Follow these steps to run this project locally.

<b>Step 1:</b> Clone the repository

```bash
git clone git@github.com:tantri-id/tantri-kitchen-display.git
```

<b>Step 2:</b> Navigate to the project directory

```bash
cd tantri-kitchen-display
```

<b>Step 3:</b> Install dependencies

<CodeGroup>

```bash npm
npm install
```

```bash yarn
yarn install
```

```bash pnpm
pnpm install
```

```bash bun
bun install
```

</CodeGroup>

<b>Step 4:</b> Create a .env file

```bash
cp .env.example .env
```

<b>Step 5:</b> Update the .env file

```bash
VITE_API_URL_DEVELOPMENT=
```

<b>Step 6:</b> Start the development server

<CodeGroup>

```bash npm
npm run dev
```

```bash yarn
yarn dev
```

```bash pnpm
pnpm run dev
```

```bash bun
bun run dev
```

</CodeGroup>

<b>Step 8:</b> Open your IDE and Start Development!
